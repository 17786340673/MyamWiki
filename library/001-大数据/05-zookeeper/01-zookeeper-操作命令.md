## 查看各个服务状态
``` sh
  bin/zkServer.sh status
```
## 重启zookeeper服务
``` sh
  bin/zkServer.sh restart
```
## 关闭zookeeper服务
``` sh
  bin/zkServer.sh stop  
```
## 启动zookeeper服务
``` sh
  bin/zkServer.sh start
```
